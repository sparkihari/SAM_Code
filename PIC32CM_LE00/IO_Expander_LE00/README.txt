 PIC32Cm LE00: IO Expander Test

 Hardware Required:

 1, PIC32CM LE00 Curiosity Board
 2, MCP23017 Breakout Board
 

 Hardware Connection:

 VCC -> 3V3/9
 GND -> GND/10
 SDA -> SDA/12
 SCL -> SCL/13



 Note: 
    
    > Add the external pull-ups if the breakout board does not have any pull-ups. 

    > All the Address lines (A0, A1, A2) are connected to VCC.
